This is a live demo from this library.
You can change interract with drop-down menus and buttons to change application's behaviour.
Demo code will be updated accordingly.

Unlike Math.random, this library allows user to choose himself a seed (both number and hashed strings), in addition to proposing differents probability distributions (you may check Documentation to get a complete list of availables distributions).

Note : Generating a large number at once may slow down the page.

    //Instantiate and seed generator
        let Distribution = Lowlight.Random.Distribution
        let generator = new Distribution.Normal().seed(0)

    //Configure generator
        generator.alpha(0).beta(0).mean(0).deviation(0).min(0).max(0).mode(null).concentration(0).scale(0).shape(0).rate(0)

    //Generate pseudo-random numbers
    //Output random number
        setInterval(function () { console.log(generator.next()) }, 500)
