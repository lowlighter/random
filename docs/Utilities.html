sample (array, length, replace, seed) → {Array}

lowlight.random.js (line 1003) 

Return a sample from an array. Original array isn't modified.
Sampled array will always be the same for a particular seed.

  • Arguments

    array Array
    String     		Array to sample
    length
    Default : array.length     		Number Sampled array length (if replace is false, sampled arrays will never be larger than original array length)
    replace
    Default : false     		Boolean If true, drawn values will be replaced in pool
    seed
    Default : false     		Number
    String
    false     		Seed (set to false to reuse previous seeded generator)

  • Returns

    Array Sampled array
